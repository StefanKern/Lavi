Welcome to LAVI

LAVI is a end to end encrypted data storage selling platform. A user can create offers for files / images groupons, basically any kind of digital data. The files, that are offered are fully encrypted on the client side. Neither the files nor any other kind of private information leaves the website. When the user uploads a file, it is already fully encrypted by a password he chooses.

For selling files, the NEO blockchain is used. It stores

  1. Open Offers (Blockhain: key: 01 + hash of file, value public key of offerer)
  2. Take Offers (Blockhain: key: 11 + hash of file, value public key of requester)
  3. Grant Offer (Blockhain: key: 21 + hash of file, value password encrypted with a shared secret)
(Data is stored as bytearrays)

The focus of this implementation was, to make the user experince as easy as possible. The user only needs to enter his WIF ONCE! The rest of the application is usable without performing any blockchain related tasks. Transactions to the blockchain are done in the background with neon-js, files are encrypted with aes on the client. The user doesn't have to upload encrypted files.

For storage purposes a SQL database in the Azure cloud is used. The blockchain cannot store large amounts of data. Complex queries which go over simple key/value pairs are also extremly hard to do. The blockchain is therefor used, for what it can do best: as a Store of Value.

The payment function is not yet included in the demo, becauuse currently NEO dAPPS create their own NEP5 tokens, when they need a currency in their contracts. This resoults in 10.000 different tokens, where only a few survive. Therefor this application LAVi aims to reuse an existing token, which is used for more then one dAPP. However currently neither NEO nor GAS nore any other toke would be suitable.
